excel-rdf ver2

Браузерные конвертеры Turtle RDF ⇄ Excel и TriG ⇄ Excel. Всё выполняется локально в браузере, файлы никуда не отправляются.

Новое в ver2: закрепление первой строки и автофильтр на всех листах, столбец «Предикаты» на листе main, поддержка именованных графов TriG.

Turtle RDF → Excel

Загрузите файл .ttl или вставьте текст Turtle, затем нажмите «Создать книгу Excel». В готовой книге на каждом листе закреплена первая строка и включён автофильтр.

Excel → Turtle RDF

Загрузите книгу .xlsx, созданную этим инструментом (или с такой же структурой листов). Источник данных для выгрузки — листы МегаТипов и лист «Прочие триплеты».

TriG → Excel

TriG — расширение Turtle с именованными графами (graph { ... }). Каждый граф раскладывается на отдельный лист (строки — субъекты, столбцы — предикаты). Соответствие листов и графов — на листе «Графы».

Excel → TriG

Загрузите книгу .xlsx со структурой листов-графов (создаётся вкладкой «TriG → Excel»). Источник данных — листы графов, перечисленные на листе «Графы».

О проекте

Редакция ver2 расширяет ver1: добавлены конвертеры TriG, закрепление первой строки и автофильтр на всех листах, а также столбец «Предикаты» на листе main. Подробности — в каталогах doc/ и analysis/.

Состав книги Excel (Turtle)

  1. main — перечень листов книги, их назначение и (для листов МегаТипов) их предикаты.
  2. Префиксы — префиксы, используемые в исходном Turtle.
  3. Turtle исходный — исходный текст Turtle RDF.
  4. Триплеты простые — по одному элементарному триплету на строку (без ; и ,).
  5. Триплеты компактные — Turtle с ; и , (сокращённое число утверждений).
  6. Листы МегаТипов — на каждый МегаТип свой лист: строки — объекты, столбцы — предикаты.
  7. Прочие триплеты — триплеты субъектов без МегаТипа.

Состав книги Excel (TriG)

  1. main — перечень листов книги, их назначение и (для листов-графов) их предикаты.
  2. Префиксы — префиксы, используемые в исходном TriG.
  3. TriG исходный — исходный текст TriG.
  4. Квадры простые — по одной четвёрке (граф, субъект, предикат, объект) на строку.
  5. Графы — соответствие листов и меток именованных графов.
  6. Листы графов — на каждый именованный граф свой лист: строки — субъекты, столбцы — предикаты.

Что нового в ver2

  • На каждом листе закреплена первая строка (freeze panes) и включён автофильтр.
  • На листе main для листов МегаТипов/графов перечислены их предикаты — наборы полей удобно сравнивать.
  • Отдельная пара конвертеров TriG → Excel и Excel → TriG с поддержкой именованных графов.
  • Детальный пример с большим числом предикатов: examples/example4.ttl.
