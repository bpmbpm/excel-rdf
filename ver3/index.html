excel-rdf ver3

Браузерные конвертеры Turtle RDF ⇄ Excel и TriG ⇄ Excel. Всё выполняется локально в браузере, файлы никуда не отправляются.

Новое в ver3: для TriG листы анализа строятся вокруг МегаТипов (по rdf:type), а принадлежность к именованному графу указывается в столбце TriG; добавлены листы «TriG с Триплеты простые» и «TriG с Триплеты компактные» и общий лист «TriG».

Turtle RDF → Excel

Загрузите файл .ttl или вставьте текст Turtle, затем нажмите «Создать книгу Excel». В готовой книге на каждом листе закреплена первая строка и включён автофильтр.

Excel → Turtle RDF

Загрузите книгу .xlsx, созданную этим инструментом (или с такой же структурой листов). Источник данных для выгрузки — листы МегаТипов и лист «Прочие триплеты».

TriG → Excel

TriG — расширение Turtle с именованными графами (graph { ... }). В ver3 субъекты раскладываются по листам МегаТипов (по значению rdf:type), а столбец TriG на листе показывает, к какому именованному графу относится утверждение. Реестр листов МегаТипов — на листе «МегаТипы», сводка по графам — на общем листе «TriG».

Excel → TriG

Загрузите книгу .xlsx со структурой листов МегаТипов (создаётся вкладкой «TriG → Excel»). Источник данных — листы МегаТипов, перечисленные на листе «МегаТипы» (граф берётся из столбца «TriG»).

О проекте

Редакция ver3 развивает ver2: для TriG листы анализа утверждений строятся вокруг МегаТипов (по rdf:type), а не вокруг графов. Принадлежность утверждения к именованному графу указывается в отдельном столбце TriG. Добавлены отдельные текстовые листы простой и компактной форм TriG и общий лист «TriG» со свойствами графов. Подробности — в каталогах doc/ и analysis/.

Состав книги Excel (Turtle)

  1. main — перечень листов книги, их назначение и (для листов МегаТипов) их предикаты.
  2. Префиксы — префиксы, используемые в исходном Turtle.
  3. Turtle исходный — исходный текст Turtle RDF.
  4. Триплеты простые — по одному элементарному триплету на строку (без ; и ,).
  5. Триплеты компактные — Turtle с ; и , (сокращённое число утверждений).
  6. Листы МегаТипов — на каждый МегаТип свой лист: строки — объекты, столбцы — предикаты.
  7. Прочие триплеты — триплеты субъектов без МегаТипа.

Состав книги Excel (TriG) — ver3

  1. main — перечень листов книги, их назначение и (для листов МегаТипов) их предикаты.
  2. Префиксы — префиксы, используемые в исходном TriG.
  3. TriG исходный — исходный текст TriG.
  4. TriG с Триплеты простые — TriG, где внутри графов по одному триплету на строку (без ; и ,).
  5. TriG с Триплеты компактные — TriG в компактной форме (с ; и ,).
  6. Квадры простые — по одной четвёрке (граф, субъект, предикат, объект) на строку.
  7. TriG — общий лист: свойства каждого графа TriG (субъекты и предикаты).
  8. МегаТипы — реестр листов МегаТипов (по rdf:type субъекта).
  9. Листы МегаТипов — на каждый МегаТип свой лист: столбцы Субъект | TriG | предикаты…; строка — пара (субъект, граф).

Что нового в ver3

  • Листы TriG строятся вокруг МегаТипов (по rdf:type), а не вокруг графов.
  • Столбец TriG на листе МегаТипа указывает именованный граф каждого утверждения.
  • Отдельные текстовые листы: TriG с Триплеты простые и TriG с Триплеты компактные.
  • Общий лист TriG собирает свойства каждого именованного графа.
  • Два примера TriG: подробный examples/trig-vad.trig и простой examples/trig-simple.trig.
  • В примере trig-vad.trig МегаТипы: ObjectTree, TypeProcess, TypeExecutor, VADProcessDia, ExecutorGroup.
