# Описание примеров

Примеры лежат в каталоге [`ver1/examples/`](../examples). Их можно загрузить во
вкладке «Turtle → Excel» (кнопка «Выбрать .ttl файл») и преобразовать в книгу Excel.

## example1.ttl — процессы и роли

Простая модель: объекты двух МегаТипов — `:Процесс` и `:Роль`.

```turtle
:Процесс_А :МегаТип :Процесс ;
    rdfs:label "Согласование договора" ;
    :владелец :Иванов ;
    :входит_в :Процесс_Б .
```

Что показывает:
- группировку субъектов по МегаТипу в отдельные листы («Процесс», «Роль»);
- предикат-ссылку на другой объект (`:владелец :Иванов`, `:входит_в :Процесс_Б`);
- литералы (`rdfs:label "..."`).

В книге появятся листы: `main`, `Префиксы`, `Turtle исходный`, `Триплеты простые`,
`Триплеты компактные`, `Процесс`, `Роль`.

## example2.ttl — несколько объектов на предикат

Модель организации: подразделения и сотрудники. Демонстрирует **списки объектов**
через запятую и несколько предикатов через точку с запятой.

```turtle
:Отдел_Продаж :МегаТип :Подразделение ;
    rdfs:label "Отдел продаж" ;
    :включает :Сидоров , :Кузнецов , :Смирнов .
```

Что показывает:
- несколько объектов одного предиката (`:включает :Сидоров , :Кузнецов , :Смирнов`) —
  в ячейке Excel они записываются через запятую;
- два МегаТипа: `:Подразделение` и `:Сотрудник`.

## example3.ttl — объекты с МегаТипом и «прочие» триплеты

Показывает, что триплеты субъектов **без** `:МегаТип` (например, описание класса
`:Товар a owl:Class`) попадают на отдельный лист **«Прочие триплеты»**, а не теряются.

```turtle
:Товар_1 :МегаТип :Товар ;
    rdfs:label "Ноутбук" ;
    :цена "75000" ;
    :категория :Электроника .

:Товар a owl:Class ;
    rdfs:label "Товар" .
```

Что показывает:
- объекты МегаТипа `:Товар` — на листе «Товар»;
- субъект `:Товар` (без `:МегаТип`) — на листе «Прочие триплеты»;
- сохранение ключевого слова `a` (rdf:type).

## Проверка round-trip

Все примеры покрыты автоматическим тестом
[`ver1/tests/test-turtle.js`](../tests/test-turtle.js): Turtle → листы книги → Turtle
не теряет триплетов и префиксов. Запуск (только для разработки, node):

```
node ver1/tests/test-turtle.js
```
